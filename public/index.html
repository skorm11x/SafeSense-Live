About this web application


This web application was developed to provide a scalable solution to SAFESENSE TECHNOLOGIES Smart Helmet needs. It allows you to access, read, and view graphical information from a database of helmet readings per category. Additionally, The web application features modes that allow you to quickly configure live test streaming, drop tests, and see relevant information in a numerical and graphical way.
